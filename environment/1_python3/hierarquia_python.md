Ferramenta	|   Função Principal
Python      |	O Interpretador (traduz o código).
Pip         |	O Gerenciador de Pacotes (instala bibliotecas como Pandas).
Pyenv(pyenv-win)       |	O Gerenciador de Versões (instala e troca o próprio Python).
Venv        |	O Ambiente Virtual (cria uma "bolha" isolada para um projeto específico).
